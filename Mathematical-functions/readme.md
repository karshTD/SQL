MySQL provides built-in mathematical functions for numerical operations directly in SQL.

Common Mathematical Functions:
Function	Description	Example	Result
ABS()	Absolute value	ABS(-10)	10
CEIL() / CEILING()	Round up to nearest integer	CEIL(4.3)	5
FLOOR()	Round down to nearest integer	FLOOR(4.7)	4
ROUND()	Round to decimal places	ROUND(123.456, 2)	123.46
POW() / POWER()	Raise to power	POW(2, 3)	8
SQRT()	Square root	SQRT(16)	4
EXP()	e raised to power	EXP(1)	2.71828
LOG()	Natural logarithm	LOG(10)	2.30259
RAND()	Random number (0-1)	RAND()	0.123456
